O que é

Fornece uma fachada de granularidade alta sobre objetos de granularidade baixa para melhorar a eficiência em uma rede.

Modelos orientados a objetos obtém melhor desempenho com objetos pequenos e com métodos pequenos. Controle de comportamentos Fácil entendimento Bastante interação entre objetos

Tais métodos não funciona bem quando executa chamadas entre processos.

Chamadas remotas tendem a ser muito custosas para se manter a alta granularidade dos objetos.

  • Preparação dos dados
  • Verificações de segurança
  • Roteamento de pacotes

Quando Usa-la

  • Sempre que precisa de acesso remoto a um modelo de objetos de baixa granularidade
  • Diferentes processos em diferentes maquinas
  • Fachadas Remotas sugerem um estilo de distribuição síncrono.
